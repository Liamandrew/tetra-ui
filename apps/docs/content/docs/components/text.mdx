title: Text
description: A lightweight wrapper around the React Native Text component to provide default styles.
---

In Uniwind and React Native, there is no way to add [base styles](https://tailwindcss.com/docs/adding-custom-styles#adding-base-styles) to all components. Therefore, we use component composition to provide a Text component that applies any necessary default styles.
